# @hound/domain-types

Shared contracts and normalization helpers for tracks, telemetry events, and orbit constants.

## Exports
- `ORBITS`
- `ROTATION_OVERRIDE`
- `PLAY_EVENT_KEYS`
- `DEFAULT_TRACK`
- `clamp(value, min, max)`
- `toISO(value)`
- `normalizeTrack(rawTrack)`
- `validatePlayEvent(event)`

## Run tests
`npm --workspace packages/domain-types run test`
